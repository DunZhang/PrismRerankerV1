# Reranker 阈值分析结果
## 方法
- 模型：voyage-rerank-2 与 voyage-rerank-2.5 得分取平均
- 数据量：38675 正例，270725 负例（1:7）
- 划分：70% 训练（KDE 拟合），30% 验证（Precision/Recall 计算）
- 先验：P(正例) = 0.1250，P(负例) = 0.8750
- 核密度估计 bandwidth：scott 方法
- 贝叶斯后验：P(正例|score) = p(score|正例) × P(正例) / [p(score|正例) × P(正例) + p(score|负例) × P(负例)]
